# Install the MRA system
1. `.claude/` folder: copy the whole folder in the root project folder.
2. `src/` folder: contains all the scripts that the system can utilize. Please make sure it is in the root folder.
3. `puzzle.md`: this is the file containing the puzzle. Make sure it is in the root folder.
4. `CLAUDE.md`: this is the file containing project-level instructions. Please make sure it is in the root folder.

## Virtual environment
- All scripts are run via virtual environment. The instructions to agents hard coded the virtual environment path which need to be replaced.
